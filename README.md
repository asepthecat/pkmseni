# 🎭 Sanggar Seni Kinanti Sekar

A modern cultural website to preserve and promote Indonesian traditional arts through an interactive digital platform.

Built with a clean frontend stack and designed to deliver an elegant, accessible, and globally-friendly user experience.

[Features](#features) | [Quick Start](#quick-start) | [Project Structure](#project-structure) | [Future Work](#future-enhancements) | [Contributing](#contributing)

---

## Why This Project

This project was developed to address the gap between traditional cultural preservation and modern digital accessibility.

It aims to:

- Bring Indonesian arts into the digital era  
- Provide an engaging platform for cultural learning  
- Reach both local and international audiences  
- Simplify access to art programs and registration  

---

## Features

- **Multi-language support** (Indonesian, English, French)  
- **Program browsing & filtering** (Dance, Music, Theatre, Craft)  
- **Detailed program modal with schedule & info**  
- **Embedded media (video trailer support)**  
- **WhatsApp integration for direct communication**  
- **Interactive chatbot interface**  
- **Responsive design for all devices**  
- **Modern UI with cultural (batik-inspired) aesthetics**
